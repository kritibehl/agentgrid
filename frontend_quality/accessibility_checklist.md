# Accessibility Checklist

## Current Checks

- Dark/light UI available
- Text labels used alongside color-coded state
- Buttons have visible text labels
- Dashboard uses semantic sections/headings
- Error and loading states use readable text
- Responsive layout stacks on small screens

## Recommended Future Checks

- Add automated axe checks
- Add keyboard focus ring audit
- Add ARIA labels for tab navigation
- Add chart descriptions for screen readers
