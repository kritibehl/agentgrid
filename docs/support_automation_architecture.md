Architecture

        ↓

        ↓
Eval Gate
        ↓
Human Review
        ↓
Feedback Loop
        ↓
Quality Metrics
Components
Component	Purpose
Issue triage	classify support issue and risk
Knowledge retrieval	fetch runbooks, policies, and evidence
Tool orchestration	call validation, metrics, and handoff tools
Eval gate	check groundedness, completeness, and safety
Human review	approve, reject, or edit risky outputs
Feedback loop	track whether the answer helped
Quality report	monitor failures and improvement opportunities
