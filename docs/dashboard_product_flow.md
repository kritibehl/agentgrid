# AgentGrid Dashboard Product Flow

## Dashboard Panels

| Panel | Purpose |
|---|---|
| Query input | User-facing support workflow entry |
| RAG trace viewer | Shows retrieved evidence sources |
| Tool-call timeline | Shows classify → retrieve → analyze → plan → eval |
| Evaluation gate panel | Shows ship/hold/escalate decision |
| AutoOps receipt | Shows incident ingestion confirmation |
| Metrics panel | Shows live Cloud Run support metrics |
| Audit/RBAC proof | Shows reviewer/admin access boundaries |

## Product Signal

The dashboard is designed to make the support workflow understandable in under 30 seconds for recruiters, support engineers, and hiring managers.
